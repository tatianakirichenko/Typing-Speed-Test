⌨️ Typing Speed Test – Тест скорости печати 
Проверьте свою скорость печати на разных языках программирования!
Пройдите тест, узнайте свою скорость в словах в минуту (WPM) и точность, улучшайте свои навыки!
Реализован на 7 языках с единым интерфейсом, выбором текстов, статистикой и цветным выводом.

🚀 Возможности
Три уровня сложности – короткий, средний и длинный текст.

Подсчёт скорости – WPM (слов в минуту) и точность в процентах.

Цветной вывод – правильные символы выделяются зелёным, ошибки – красным.

Выбор текста – случайный выбор из встроенного набора или указание пользователем.

Статистика – сохраняет лучший результат в локальный файл.

Интерактивность – ввод текста с таймером, отображение прогресса.

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> [уровень] [опции]
Уровни сложности
easy – короткий текст (~20 слов)

medium – средний текст (~40 слов)

hard – длинный текст (~60 слов)

Если уровень не указан, выбирается medium.

Опции
Опция	Описание
-t, --text <текст>	Использовать свой текст для теста
-s, --stats	Показать сохранённую статистику
-r, --reset	Сбросить статистику
-h, --help	Справка
Примеры
bash
# Тест на среднем уровне
python typing.py medium

# Тест с собственным текстом
python typing.py -t "Мой текст для проверки скорости"

# Показать статистику
python typing.py -s
Управление
Во время теста введите текст и нажмите Enter.

После завершения будет показан результат.

q – выход во время теста.

🧠 Логика работы
Выбор текста – случайный из встроенного словаря или пользовательский.

Отображение – текст выводится на экран.

Таймер – время начинается с момента ввода первой буквы.

Ввод – пользователь вводит текст и нажимает Enter.

Сравнение – подсчитывается количество правильных символов, ошибок, общее время.

Расчёт – WPM (количество слов ÷ время в минутах), точность (правильные символы / общее количество).

Сохранение – лучший результат записывается в файл.

✨ Дополнительные фичи
Цветная подсветка – при сравнении показываются ошибки.

Сохранение статистики – в ~/.typing_stats.json хранятся лучшие результаты для каждого уровня.

Гибкий словарь – можно легко добавить свои тексты.
